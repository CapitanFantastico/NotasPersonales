## Proceso para pintar a mano con un tono natural que realce las vetas de la madera
- Mise en place (Alistamiento)
	- Ropa de trabajo (incluye zapatos y gorra)
	- Elementos de seguridad personal
		- Tapabocas
		- Gafas
		- Guantes de látex
		- Cuarto de lavado
	- Mantel plástico
	- Mantel de papel
	- superficie de madera - mártir 
	- Mecha para limpiar
	- Thinner
	- Vaso desechables para mezclas
	- Vaso desechable para lavado
	- Palitos de pincho para revolver
	- Bandeja de pinturas y diluyentes
	- Brocha
	- Piezas de madera, tornillos, puntillas y cordel para colgar
	- Reconocer espacio de trabajo para pintar y secar
	- Disponer de todas las pinturas y diluyentes
- Cepillar
- Lijar 100
- Inmunizar
	- Usar lorsban
	- Aplicarlo a la sombra en una zona aireada
- Resanar
	- Tener buena iluminación para detectar fallas
	- El estuco plástico no es muy resistente, pero se puede usar
	- Nunca usar resane con colbón, no se puede pintar luego
	- Puede usarse huesoduro o invecryl (resina acrílica impermeabilizante)
	- Mezclar sellador M40 o laca Nitro con polvillo limpio de madera
- Lijar 100
- Sellar (2 capas)
	- Usar sellador M40, disolver con thinner a ojo
	- Aplicar tinte aquí si se desea
	- Aplicar la primera capa gruesa con brocha
	- Dejar secar
	- Lijar 220
	- Aplicar una segunda capa
	- Dejar secar
	- Lijar con la misma lija usada antes
- Tinte
	- Si no se aplicó tinte con el sellador, aquí se puede aplicar.  El tinte se diluye con alcohol industrial 90 grados y este es costoso
- Esmalte base  
	- Deja ver la veta de la madera y se usa en lugar de tintes
	- No se puede combinar con otras pinturas 
	- Disolver con varsol
	- El caoba es el más natural, se puede adicionar tinte (ocre, rojo, almendra) para modificar el color
	- Se aplica con brocha y se limpia con wiper (trapo)
- Lacar (2 capas)
	- Tener buena iluminación para detectar fallas
	- Aplicar con brocha laca Nitro, disolver con thinner a ojo
	- Aplicar tinte aquí si se desea
	- Dejar secar
	- Lijar 400
	- Aplicar una segunda capa
	- Dejar secar

- Envejecido
	- Aplicar la dilusión de alquitrán en gasolina o varsol.
	- El alquitrán inmuniza e impermeabiliza
	- Se puede agregar tinte
